# **A/B Testing Portfolio**

Этот репозиторий содержит различные проекты, основанные на **A/B тестировании**, с использованием реальных и симулированных данных. В каждом проекте применяются статистические методы для проверки гипотез и оценки эффективности изменений.

### **Используемые инструменты:**

* Python (Pandas, NumPy, SciPy, Statsmodels)
* Jupyter Notebooks
* Статистические тесты (Z-тест, t-тест, ANOVA, Bootstrap)

### **Цель проектов:**

Продемонстрировать навыки в **проведении экспериментов**, анализе данных и **интерпретации статистических результатов**.
